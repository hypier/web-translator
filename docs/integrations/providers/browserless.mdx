# Browserless

>[Browserless](https://www.browserless.io/docs/start) 是一个允许您在云中运行无头 Chrome 实例的服务。这是一个在不必担心管理自己基础设施的情况下，以规模运行基于浏览器的自动化的好方法。

## 安装与设置

我们需要在[这里](https://www.browserless.io/pricing/)获取API密钥。
