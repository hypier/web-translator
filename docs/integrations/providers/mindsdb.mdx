# MindsDB

MindsDB 是一个用于从企业数据定制 AI 的平台。借助 MindsDB 及其近 200 个与 [数据源](https://docs.mindsdb.com/integrations/data-overview) 和 [AI/ML 框架](https://docs.mindsdb.com/integrations/ai-overview) 的集成，任何开发者都可以利用他们的企业数据快速且安全地定制 AI 以满足他们的需求。

使用 MindsDB，您可以将任何数据源连接到任何 AI/ML 模型，以实施和自动化 AI 驱动的应用程序。实时部署、服务和微调模型，利用来自数据库、向量存储或应用程序的数据。使用开发者已经熟悉的通用工具完成所有这些操作。

MindsDB 与 LangChain 集成，使用户能够：

- 在 MindsDB 中部署通过 LangChain 提供的模型，使其可访问多个数据源。
- 使用实时和动态数据在 MindsDB 中微调通过 LangChain 提供的模型。
- 使用 LangChain 和 MindsDB 自动化 AI 工作流。

请访问 [我们的文档](https://docs.mindsdb.com/integrations/ai-engines/langchain) 以了解更多关于 MindsDB 与 LangChain 集成的信息，并查看示例。
