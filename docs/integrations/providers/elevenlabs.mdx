# ElevenLabs

>[ElevenLabs](https://elevenlabs.io/about) 是一家语音 AI 研究与部署公司，旨在使内容在任何语言和声音中都能普遍可及。
>
>`ElevenLabs` 创建了最逼真、多功能且具上下文意识的 AI 音频，能够在 29 种语言中生成数百种新旧声音的语音。

## 安装与设置

首先，您需要设置一个 ElevenLabs 账户。您可以按照 
[这里的说明](https://docs.elevenlabs.io/welcome/introduction)进行操作。

安装 Python 包：

```bash
pip install elevenlabs
```

## 工具

查看 [使用示例](/docs/integrations/tools/eleven_labs_tts)。
