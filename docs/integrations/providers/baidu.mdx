# 百度

>[百度云](https://cloud.baidu.com/) 是由 `Baidu, Inc.` 提供的云服务， 
> 总部位于北京。它提供云存储服务、客户端软件、 
> 文件管理、资源共享和第三方集成。

## 安装与设置

在[这里](https://cloud.baidu.com/product/wenxinworkshop)注册并获取`Qianfan` `AK`和`SK`密钥。

## LLMs

### 百度千帆

查看[用法示例](/docs/integrations/llms/baidu_qianfan_endpoint)。

```python
from langchain_community.llms import QianfanLLMEndpoint
```
