# Psychic

:::warning
此提供者不再维护，可能无法正常工作。请谨慎使用。
:::

>[Psychic](https://www.psychic.dev/) 是一个与 `Notion`、`Zendesk`、`Confluence` 和 `Google Drive` 等 SaaS 工具通过 OAuth 集成的平台，并将这些应用程序中的文档同步到您的 SQL 或向量数据库。您可以将其视为用于非结构化数据的 Plaid。

## 安装与设置

```bash
pip install psychicapi
```

Psychic 的设置非常简单 - 你只需导入 `react` 库，并使用你的 `Sidekick API` 密钥进行配置，该密钥可以从 [Psychic 仪表盘](https://dashboard.psychic.dev/) 获取。当你连接应用程序时，你可以从仪表盘查看这些连接，并使用服务器端库检索数据。

1. 在 [仪表盘](https://dashboard.psychic.dev/) 创建一个账户。
2. 使用 [react 库](https://docs.psychic.dev/sidekick-link) 将 Psychic 链接模态添加到你的前端 react 应用中。你将用它来连接 SaaS 应用。
3. 一旦你创建了连接，你可以通过遵循 [示例笔记本](/docs/integrations/document_loaders/psychic) 使用 `PsychicLoader`。

## 优势与其他文档加载器的比较

1.	**通用 API:** 不必为每个 SaaS 应用构建 OAuth 流程和学习 API，您只需集成 Psychic 一次，即可利用我们的通用 API 来检索数据。
2.	**数据同步:** 客户的 SaaS 应用中的数据可能会迅速过时。使用 Psychic，您可以配置 Webhook，以确保您的文档每天或实时保持最新。
3.	**简化 OAuth:** Psychic 处理端到端的 OAuth，因此您不必花时间为每个集成创建 OAuth 客户端、保持访问令牌的新鲜和处理 OAuth 重定向逻辑。
