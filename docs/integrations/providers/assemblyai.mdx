# AssemblyAI

>[AssemblyAI](https://www.assemblyai.com/) 构建 `Speech AI` 模型，用于语音转文本、说话人分离、语音摘要等任务。
> `AssemblyAI` 的 Speech AI 模型包括针对语音数据（如电话、虚拟会议和播客）的准确语音转文本、说话人检测、情感分析、章节检测和个人身份信息（PII）编辑。

## 安装与设置

获取您的 [API 密钥](https://www.assemblyai.com/dashboard/signup)。

安装 `assemblyai` 包。

```bash
pip install -U assemblyai
```

## 文档加载器

### AssemblyAI音频转录

`AssemblyAIAudioTranscriptLoader` 使用 `AssemblyAI API` 转录音频文件，并将转录的文本加载到文档中。

查看 [使用示例](/docs/integrations/document_loaders/assemblyai)。

```python
from langchain_community.document_loaders import AssemblyAIAudioTranscriptLoader
```
