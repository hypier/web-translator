# Meilisearch

> [Meilisearch](https://meilisearch.com) 是一个开源、快速且高度相关的搜索引擎。
> 它提供了优秀的默认设置，帮助开发者构建流畅的搜索体验。
>
> 您可以 [自托管 Meilisearch](https://www.meilisearch.com/docs/learn/getting_started/installation#local-installation) 
> 或者在 [Meilisearch Cloud](https://www.meilisearch.com/pricing) 上运行。
>
>`Meilisearch v1.3` supports vector search.

## 安装与设置

有关详细配置说明，请参见[使用示例](/docs/integrations/vectorstores/meilisearch)。

我们需要安装 `meilisearch` python 包。

```bash
pip install meilisearch
```
