# Notion DB

>[Notion](https://www.notion.so/) 是一个协作平台，支持修改过的 Markdown，集成了看板、任务、维基和数据库。它是一个集笔记、知识和数据管理以及项目和任务管理于一体的工作空间。

## 安装与设置

所有说明都在下面的示例中。

## 文档加载器

我们有两种不同的加载器：`NotionDirectoryLoader` 和 `NotionDBLoader`。

请查看 [NotionDirectoryLoader 的使用示例](/docs/integrations/document_loaders/notion)。

```python
from langchain_community.document_loaders import NotionDirectoryLoader
```
