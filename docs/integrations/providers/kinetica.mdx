# Kinetica

[Kinetica](https://www.kinetica.com/) 是一个实时数据库，专为在时间序列和空间数据上实现分析和生成性人工智能而构建。

## 聊天模型

Kinetica LLM 封装使用 [Kinetica SqlAssist LLM](https://docs.kinetica.com/7.2/sql-gpt/concepts/) 将自然语言转换为 SQL，以简化数据检索过程。

有关用法，请参见 [Kinetica SqlAssist LLM 演示](/docs/integrations/chat/kinetica)。

```python
from langchain_community.chat_models.kinetica import ChatKinetica
```

## 向量存储

Kinetca 向量存储封装利用了 Kinetica 对 [向量相似性搜索](https://docs.kinetica.com/7.2/vector_search/) 的原生支持。

请参阅 [Kinetica Vectorsore API](/docs/integrations/vectorstores/kinetica) 以获取使用说明。

```python
from langchain_community.vectorstores import Kinetica
```

## 文档加载器

Kinetica 文档加载器可用于从 Kinetica 数据库加载 LangChain 文档。

请参阅 [Kinetica 文档加载器](/docs/integrations/document_loaders/kinetica) 以获取用法

```python
from langchain_community.document_loaders.kinetica_loader import KineticaLoader
```

## Retriever

Kinetica Retriever 可以根据非结构化查询返回文档。

请参阅 [Kinetica VectorStore 基于 Retriever 的使用](/docs/integrations/retrievers/kinetica)。
