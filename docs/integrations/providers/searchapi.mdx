# SearchApi

本页面介绍如何在 LangChain 中使用 [SearchApi](https://www.searchapi.io/) Google Search API。SearchApi 是一个实时的 SERP API，便于进行 SERP 抓取。

## 设置

- 访问 [https://www.searchapi.io/](https://www.searchapi.io/) 注册一个免费账户
- 获取 api 密钥并将其设置为环境变量 (`SEARCHAPI_API_KEY`)

## 包装器

### 工具

有一个名为 SearchApiAPIWrapper 的工具，它封装了这个 API。要导入这个工具：

```python
from langchain_community.utilities import SearchApiAPIWrapper
```

您可以将其作为 Self Ask 链的一部分使用：

```python
from langchain_community.utilities import SearchApiAPIWrapper
from langchain_openai import OpenAI
from langchain.agents import initialize_agent, Tool
from langchain.agents import AgentType

import os

os.environ["SEARCHAPI_API_KEY"] = ""
os.environ['OPENAI_API_KEY'] = ""

llm = OpenAI(temperature=0)
search = SearchApiAPIWrapper()
tools = [
    Tool(
        name="中间答案",
        func=search.run,
        description="在需要通过搜索提问时非常有用"
    )
]

self_ask_with_search = initialize_agent(tools, llm, agent=AgentType.SELF_ASK_WITH_SEARCH, verbose=True)
self_ask_with_search.run("谁活得更久：柏拉图、苏格拉底还是亚里士多德？")
```

### 工具

您也可以轻松地将这个包装器加载为工具（以便与代理一起使用）。
您可以这样做：

```python
from langchain.agents import load_tools
tools = load_tools(["searchapi"])
```

有关工具的更多信息，请参见[此页面](/docs/how_to/tools_builtin)。
