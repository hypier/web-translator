# DingoDB

本页面介绍如何在 LangChain 中使用 DingoDB 生态系统。
内容分为两个部分：安装和设置，以及对特定 DingoDB 包装器的参考。

## 安装与设置
- 使用 `pip install dingodb` 安装 Python SDK

## VectorStore

存在一个围绕 DingoDB 索引的封装，使您可以将其用作向量存储，无论是用于语义搜索还是示例选择。

要导入此向量存储：
```python
from langchain_community.vectorstores import Dingo
```

有关 DingoDB 封装的更详细说明，请参见 [此笔记本](/docs/integrations/vectorstores/dingo)
