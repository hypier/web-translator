# Apache Doris

>[Apache Doris](https://doris.apache.org/) 是一个用于实时分析的现代数据仓库。
它能够对大规模的实时数据进行闪电般快速的分析。

>通常 `Apache Doris` 被归类为 OLAP，并且在 [ClickBench — 一个分析型数据库基准测试](https://benchmark.clickhouse.com/) 中表现出色。
>由于它具有超快的向量化执行引擎，因此也可以作为快速的向量数据库使用。

## 安装与设置

```bash
pip install pymysql
```
