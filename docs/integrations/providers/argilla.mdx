# Argilla

>[Argilla](https://argilla.io/) 是一个开源的数据管理平台，专为 LLMs 设计。 
> 使用 `Argilla`，每个人都可以通过更快的数据管理，结合人类和机器反馈，构建强大的语言模型。 `Argilla` 在 MLOps 生命周期的每个步骤中提供支持，从数据标注到模型监控。

## 安装和设置

获取您的 [API 密钥](https://platform.openai.com/account/api-keys)。

安装 Python 包：

```bash
pip install argilla
```

## 回调


```python
from langchain.callbacks import ArgillaCallbackHandler
```
