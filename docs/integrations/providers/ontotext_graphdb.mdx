# Ontotext GraphDB

>[Ontotext GraphDB](https://graphdb.ontotext.com/) 是一个符合 RDF 和 SPARQL 的图形数据库和知识发现工具。

## 依赖项

安装 [rdflib](https://github.com/RDFLib/rdflib) 包，使用
```bash
pip install rdflib==7.0.0
```

## 图形问答链

将您的 GraphDB 数据库与聊天模型连接，以获取关于您数据的见解。

请参阅笔记本示例 [这里](/docs/integrations/graphs/ontotext)。

```python
from langchain_community.graphs import OntotextGraphDBGraph
from langchain.chains import OntotextGraphDBQAChain
```
