# Hazy Research

本页面介绍如何在 LangChain 中使用 Hazy Research 生态系统。
它分为两个部分：安装和设置，以及对特定 Hazy Research 包装器的引用。

## 安装与设置
- 要使用 `manifest`，请使用 `pip install manifest-ml` 进行安装

## 包装器

### LLM

存在一个围绕 Hazy Research 的 `manifest` 库的 LLM 包装器。 
`manifest` 是一个 Python 库，它本身是许多模型提供者的包装器，并添加了缓存、历史记录等功能。

要使用此包装器：
```python
from langchain_community.llms.manifest import ManifestWrapper
```
