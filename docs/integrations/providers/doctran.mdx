# Doctran

>[Doctran](https://github.com/psychic-api/doctran) 是一个 Python 包。它利用 LLMs 和开源 NLP 库将原始文本转换为干净、结构化、信息密集的文档，这些文档经过优化以便于向量空间检索。你可以将 `Doctran` 想象成一个黑匣子，杂乱的字符串输入后，会输出整洁、干净、带标签的字符串。

## 安装与设置

```bash
pip install doctran
```
