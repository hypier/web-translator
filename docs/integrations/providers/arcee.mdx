# Arcee

>[Arcee](https://www.arcee.ai/about/about-us) 使得我们所称的 SLM（小型、专业、安全和可扩展的语言模型）的开发和进步成为可能。通过提供 SLM 适配系统和无缝、安全的集成，`Arcee` 使企业能够充分利用领域适应的语言模型的潜力，推动运营中的变革性创新。

## 安装与设置

获取您的 `Arcee API` 密钥。

## LLMs

查看 [用法示例](/docs/integrations/llms/arcee)。

```python
from langchain_community.llms import Arcee
```
