# SerpAPI

本页介绍如何在 LangChain 中使用 SerpAPI 搜索 API。内容分为两个部分：安装和设置，以及对特定 SerpAPI 包装器的参考。

## 安装与设置
- 使用 `pip install google-search-results` 安装依赖
- 获取 SerpAPI 的 API 密钥，并将其设置为环境变量 (`SERPAPI_API_KEY`)

## 包装器

### 实用工具

存在一个 SerpAPI 工具，它封装了这个 API。要导入这个工具：

```python
from langchain_community.utilities import SerpAPIWrapper
```

有关此包装器的更详细说明，请参见 [这个笔记本](/docs/integrations/tools/serpapi)。

### 工具

您也可以轻松地将此包装器加载为工具（与代理一起使用）。  
您可以使用以下代码做到这一点：  
```python
from langchain.agents import load_tools
tools = load_tools(["serpapi"])
```

有关更多信息，请参见 [此页面](/docs/how_to/tools_builtin)
