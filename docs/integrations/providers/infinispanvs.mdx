# Infinispan VS

> [Infinispan](https://infinispan.org) Infinispan 是一个开源的内存数据网格，提供能够存储所有类型数据的键值数据存储，从 Java 对象到纯文本。自 15 版本以来，Infinispan 支持对缓存进行向量搜索。

## 安装与设置
请参阅 [入门指南](https://infinispan.org/get-started/) 以运行 Infinispan 服务器，您可能希望禁用身份验证
（目前不支持）
