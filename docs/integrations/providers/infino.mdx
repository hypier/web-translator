# Infino

>[Infino](https://github.com/infinohq/infino) 是一个开源的可观察性平台，可以同时存储指标和应用日志。

`Infino` 的主要功能包括：
- **指标跟踪**：捕获 LLM 模型处理请求所需的时间、错误、令牌数量以及特定 LLM 的成本指示。
- **数据跟踪**：记录并存储每次 LangChain 交互的提示、请求和响应数据。
- **图形可视化**：生成基本的时间图，描绘请求持续时间、错误发生次数、令牌计数和成本等指标。

## 安装和设置

首先，您需要按照以下步骤安装 `infinopy` Python 包：

```bash
pip install infinopy
```

如果您已经有运行中的 `Infino Server`，那么您可以继续；但如果没有，请按照以下步骤启动它：

- 确保您已安装 Docker
- 在终端中运行以下命令：
    ```
    docker run --rm --detach --name infino-example -p 3000:3000 infinohq/infino:latest
    ```
