# Qdrant

>[Qdrant](https://qdrant.tech/documentation/)（读作：四分之一）是一个向量相似性搜索引擎。 
> 它提供一个生产就绪的服务，具有方便的 API 用于存储、搜索和管理 
> 点 - 带有额外负载的向量。`Qdrant` 针对扩展过滤支持进行了定制。

## 安装与设置

安装 Python 合作伙伴包：

```bash
pip install langchain-qdrant
```

## 向量存储

存在一个围绕 `Qdrant` 索引的包装器，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

要导入此向量存储：
```python
from langchain_qdrant import QdrantVectorStore
```

有关 Qdrant 包装器的更详细说明，请参见 [此笔记本](/docs/integrations/vectorstores/qdrant)
