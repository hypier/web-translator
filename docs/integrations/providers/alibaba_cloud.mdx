# Alibaba Cloud

>[Alibaba Group Holding Limited (Wikipedia)](https://en.wikipedia.org/wiki/Alibaba_Group)，或称`Alibaba`
>（中文：阿里巴巴），是一家专注于电子商务、零售、互联网和技术的中国跨国科技公司。
> 
>[Alibaba Cloud (Wikipedia)](https://en.wikipedia.org/wiki/Alibaba_Cloud)，也称为`Aliyun`
>（中文：阿里云；拼音：Ālǐyún；意为“阿里云”），是一家云计算公司，是`Alibaba Group`的子公司。`Alibaba Cloud`为在线业务和阿里巴巴自身的电子商务生态系统提供云计算服务。
