# SparkLLM

>[SparkLLM](https://xinghuo.xfyun.cn/spark) 是由 iFLYTEK 独立开发的大规模认知模型。它通过学习大量文本、代码和图像，具备跨领域的知识和语言理解能力。它能够理解并根据自然对话执行任务。
