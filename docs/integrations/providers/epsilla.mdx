# Epsilla

本页面介绍如何在 LangChain 中使用 [Epsilla](https://github.com/epsilla-cloud/vectordb)。内容分为两个部分：安装与设置，以及对特定 Epsilla 包装器的引用。

## 安装与设置

- 使用 `pip/pip3 install pyepsilla` 安装 Python SDK

## 包装器

### VectorStore

存在一个围绕 Epsilla 向量数据库的封装，使您可以将其用作向量存储，无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain_community.vectorstores import Epsilla
```

有关 Epsilla 封装的更详细说明，请参见 [此笔记本](/docs/integrations/vectorstores/epsilla)
