# Salute Devices

Salute Devices 提供 GigaChat LLM 的模型。

有关如何访问 GigaChat 的更多信息，请 [点击这里](https://developers.sber.ru/docs/ru/gigachat/api/integration).

## 安装与设置

GigaChat 包可以通过 pip 从 PyPI 安装：

```bash
pip install gigachat
```

## LLMs

查看[使用示例](/docs/integrations/llms/gigachat)。

```python
from langchain_community.llms import GigaChat
```

## 聊天模型

参见 [使用示例](/docs/integrations/chat/gigachat).

```python
from langchain_community.chat_models import GigaChat
```

## 嵌入

查看 [用法示例](/docs/integrations/text_embedding/gigachat).

```python
from langchain_community.embeddings import GigaChatEmbeddings
```
