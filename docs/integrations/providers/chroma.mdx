# Chroma

>[Chroma](https://docs.trychroma.com/getting-started) 是一个用于构建带有嵌入的 AI 应用程序的数据库。

## 安装与设置

```bash
pip install langchain-chroma
```

## VectorStore

存在一个围绕 Chroma 向量数据库的封装，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

```python
from langchain_chroma import Chroma
```

有关 Chroma 封装的更详细的操作指南，请参见 [此笔记本](/docs/integrations/vectorstores/chroma)
