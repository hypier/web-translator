# Brave Search

>[Brave Search](https://en.wikipedia.org/wiki/Brave_Search) 是由 Brave Software 开发的搜索引擎。
> - `Brave Search` 使用自己的网络索引。截至 2022 年 5 月，它覆盖了超过 100 亿个页面，并在不依赖任何第三方的情况下提供 92% 的搜索结果，其余的则通过 Bing API 从服务器端检索，或者（在用户选择的情况下）从 Google 客户端检索。根据 Brave 的说法，索引被故意保持在“比 Google 或 Bing 小”的水平，以帮助避免垃圾信息和其他低质量内容，缺点是“Brave Search 在检索长尾查询方面尚不如 Google。”
>- `Brave Search Premium`：截至 2023 年 4 月，Brave Search 是一个无广告的网站，但最终将转向一个新的模式，届时将包含广告，付费用户将获得无广告体验。默认情况下不会收集用户数据，包括 IP 地址。需要一个高级账户才能选择数据收集。

## 安装与设置

要访问 Brave Search API，您需要 [创建一个帐户并获取 API 密钥](https://api.search.brave.com/app/dashboard)。
