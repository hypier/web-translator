# Beautiful Soup

>[Beautiful Soup](https://www.crummy.com/software/BeautifulSoup/) 是一个用于解析 HTML 和 XML 文档的 Python 包（包括处理格式不正确的标记，即未闭合的标签，因此被称为标签汤）。它为解析后的页面创建解析树，可以用于从 HTML 中提取数据，这对于网络爬虫非常有用。

## 安装和设置

```bash
pip install beautifulsoup4
```
