# Activeloop Deep Lake

>[Activeloop Deep Lake](https://docs.activeloop.ai/) 是一个用于深度学习应用的数据湖，允许您将其用作向量存储。

## 为什么选择 Deep Lake？

- 不仅仅是一个（多模态）向量存储。您可以稍后使用该数据集来微调自己的 LLM 模型。
- 不仅存储嵌入，还自动管理原始数据的版本控制。
- 真正的无服务器。无需其他服务，可以与主要云服务提供商（`AWS S3`、`GCS` 等）一起使用。

`Activeloop Deep Lake` 支持 `SelfQuery Retrieval`：
[Activeloop Deep Lake Self Query Retrieval](/docs/integrations/retrievers/self_query/activeloop_deeplake_self_query)
