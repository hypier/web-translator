# LanceDB

本页介绍如何在 LangChain 中使用 [LanceDB](https://github.com/lancedb/lancedb)。内容分为两个部分：安装与设置，以及对特定 LanceDB 包装器的引用。

## 安装与设置

- 使用 `pip install lancedb` 安装 Python SDK

## 包装器

### VectorStore

存在一个围绕LanceDB数据库的封装，允许您将其用作向量存储，无论是用于语义搜索还是示例选择。

要导入此向量存储：

```python
from langchain_community.vectorstores import LanceDB
```

有关LanceDB封装的更详细说明，请参见[此笔记本](/docs/integrations/vectorstores/lancedb)
