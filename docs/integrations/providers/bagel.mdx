# 贝果

> [贝果](https://www.bagel.net/) (`Open Vector Database for AI`) 是一个类似于 GitHub 的 AI 数据平台。
它是一个协作平台，用户可以创建、分享和管理向量数据集。它可以支持独立开发者的私人项目、企业内部的协作以及数据 DAO 的公开贡献。

## 安装与设置

```bash
pip install bagelML
```
