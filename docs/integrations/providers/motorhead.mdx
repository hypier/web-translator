# Motörhead

>[Motörhead](https://github.com/getmetal/motorhead) 是一个用 Rust 实现的内存服务器。它在后台自动处理增量摘要，并允许无状态应用程序。

## 安装与设置

请参阅 [Motörhead](https://github.com/getmetal/motorhead) 上的说明，以在本地运行服务器。
