# 常见问题解答

## 拉取请求 (PRs)

### 如何允许维护者编辑我的 PR？

当您提交拉取请求时，合并之前可能需要进行额外的更改。通常情况下，维护者自己进行这些更改会比在代码审查中要求您进行更改更高效。

默认情况下，大多数拉取请求在右侧边栏会有一个 
`✅ Maintainers are allowed to edit this pull request.`
徽章。

如果您没有看到这个徽章，您可能在您所提交拉取请求的 fork 中关闭了此设置。有关更多信息，请参见 [这篇 Github 文档](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/working-with-forks/allowing-changes-to-a-pull-request-branch-created-from-a-fork)。

值得注意的是，Github 不允许在 **组织** 的 fork 中启用此设置 ([issue](https://github.com/orgs/community/discussions/5634))。如果您在一个组织中工作，我们建议您从个人 fork 提交您的 PR，以便启用此设置。
